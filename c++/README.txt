To Compile the programs type 'make' in terminal.

To run dining_philosiphers program
./dining_philosiphers "number_of_diners"
eg: ./dining_philosiphers 10

To run reader_writers program
./reader_writers "number_of_reader" "number_of_writers"
eg: ./reader_writers 10

To run sleeping_barber program
./sleeping_barber "number_of_chairs" "number_of_customers"
eg: ./sleeping_barber 10

To run bounded_buffer program
./bounded_buffer "number_of_elements_to_produce"
eg: ./bounded_buffer 10
